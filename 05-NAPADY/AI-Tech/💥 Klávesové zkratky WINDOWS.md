[[KLÁVESOVÉ ZKRATKY]]


> [!warning]+ **KLÁVESOVÉ ZKRATKY**
> - cmd/ctrl + k ---> přidat odkaz k textu
> - add property ... cmd + ;  | alt + P //mac
> - obrácený apostrof ||| PRAVÝ ALT + 7
> - ^ ... "stříška" pr-alt + ž //mac
> - alt+X = hashtag //win
> - alt+3 = hashtag //mac
> - atl+91,93= [hranaté závorky]



| Jak pořídit snímek | Kdy se to hodí?                                                                                         |
| ------------------ | ------------------------------------------------------------------------------------------------------- |
| PrintScreen        | Když chcete snímek celé obrazovky.                                                                      |
| Win + Shift + S    | Když chcete výřez obrazovky, když chcete soubor rychle upravit nebo uložit (pomocí nástroje Výstřižky). |
| Win + Alt + R      | Když chcete nahrát videozáznam obrazovky (pomocí nástroje Xbox Bar).                                    |
