# Creative talks website is an inspiration in itself

* 1
	[[#|inShare]]

Featuring vibrant colours and brilliant content, the redesign of Tina Roth Eisenberg's CreativeMornings site both inspires and informs.

![[./_resources/Creative_talks_website_is_an_inspiration_in_itself__Web_design__Creative_Bloq.resources/cm1.jpg]]

[CreativeMornings](http://creativemornings.com/) was started in 2008 by [Tina Roth Eisenberg](https://twitter.com/swissmiss) as a simple monthly morning talk and meetup event, including breakfast free of charge. CreativeMornings has continued to grow around the world and, late last year, launched a new website.

Built by New York-based [Oak studios](http://oak.is/), the website is clean and easy to use. JavaScript features heavily. The site also uses Ruby for the backend and SCSS, Haml and CoffeeScript for the frontend.

![[./_resources/Creative_talks_website_is_an_inspiration_in_itself__Web_design__Creative_Bloq.resources/cm2.jpg]]

CreativeMornings new site is clean and easy to use

“SCSS allows us to store commonly used colours and dimensions as variables, so if we ever want to change something, we only have to change one line,” says partner and frontend developer sawyer Hollenshead. “We also use SCSS mixins and functions for responsive styles throughout the entire site,” he adds.

The video content is controlled via YouTube’s IFrame player API with the atrament web and Proxima nova fonts delivered by [Typekit](https://typekit.com/). Some new HTML5 elements are also used; it’s good to see the <time> element being used with the datetime attribute.
