# Eisenhower Matrix Teplate

* * *

About this template:
Prioritize all your tasks and to-dos into one of four quadrants in this ‘[Eisenhower Matrix](https://evernote.com/blog/work-effectively-with-the-eisenhower-matrix/)’. Think of a task you need to do today. How do you decide when you’ll get it done, given all the competition from other items on your to-do list? Use the Eisenhower Matrix to help you figure it out. Where you decide your task falls within a specific quadrant dictates where, when, and how long you should take to do that task..

* * *

|     |     |
| --- | --- |
| ### URGENT + IMPORTANT<br><br>### _Do it now_ | ### IMPORTANT, NOT URGENT<br><br>### _Decide when to do it_ |
| * [ ] Task 1<br><br>* [ ] Task 2<br><br>* [ ] Task 3 | * [ ] Task 1 — do on \[DATE\]<br><br>* [ ] Task 2 — do on \[DATE\]<br><br>* [ ] Task 3 — do on \[DATE\] |
| ### URGENT, NOT IMPORTANT<br><br>### _Delegate_ | ### NOT URGENT, NOT IMPORTANT<br><br>### _Do it later / Dump it_ |
| * [ ] Task 1 — delegate to \[NAME\]<br><br>* [ ] Task 2 — delegate to \[NAME\]<br><br>* [ ] Task 3 — delegate to \[NAME\] | * [ ] Idea 1<br><br>* [ ] Idea 2<br><br>* [ ] Idea 3 |
