Vytvoř jednoduchý, hezký a barevný jednostránkový HTML web pro půjčovnu paddleboardů na jezeře Milada v Ústeckém kraji.

DŮLEŽITÉ:
- Výstup musí být 1 samostatný HTML soubor.
- Použij čisté HTML + interní CSS + případně malý JavaScript.
- Nepoužívej frameworky.
- Web musí vypadat moderně, letně, svěže a důvěryhodně.
- Barevnost: tyrkysová, modrá, bílá, žluté/oranžové akcenty.
- Styl: voda, léto, pohyb, relax, rodiny, školy.
- Web má být responzivní a dobře čitelný na mobilu.
- Texty piš česky.
- Strukturu drž jednoduchou a přehlednou.

ÚČEL WEBU:
- představit půjčovnu paddleboardů na Miladě
- oslovit veřejnost i školy
- podpořit rezervace a kontakt
- ukázat ceník a důvody, proč přijet právě sem
