### Co je Back End

**Back End** je serverová část webové aplikace, která zpracovává logiku aplikace, správu databáze, autentizaci uživatelů, autorizaci a další serverové procesy. Back End může být napsán v různých programovacích jazycích, jako jsou JavaScript (Node.js), Python (Django, Flask), Ruby (Ruby on Rails), PHP, Java atd., a komunikuje s databázemi jako MySQL, PostgreSQL, MongoDB, atd.

Back End obvykle obsahuje:

- **Server**: Poslouchá požadavky přicházející z internetu a odpovídá na ně.
- **Aplikační logiku**: Kód, který určuje, co aplikace dělá s přijatými požadavky, jak zpracovává data atd.
- **Databázi**: Místo pro ukládání a získávání dat potřebných pro aplikaci.
