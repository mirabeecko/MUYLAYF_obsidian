React je otevřená JavaScriptová knihovna pro vytváření uživatelských rozhraní, zejména pro jednostránkové aplikace. Byl vyvinut a udržován Facebookem spolu s komunitou individuálních vývojářů a společností. React umožňuje vývojářům vytvářet velké webové aplikace, které mohou měnit data, aniž by bylo nutné znovu načítat stránku.

##### K čemu se používá?

1. **Vytváření dynamických webových aplikací:** React usnadňuje vytváření interaktivních UI a zajišťuje efektivní aktualizaci a vykreslování komponent při změně dat.
 2. **Komponentní architektura:** Umožňuje vývojářům stavět aplikace z izolovaných, znovupoužitelných komponent, což zvyšuje účinnost a usnadňuje správu kódu.
 3. **Jednosměrný tok dat:** Zjednodušuje složitost správy dat a pomáhá vytvářet stabilnější a předvídatelnější aplikace.

##### Nejznámější programy využívající React

 1. **Facebook:** Jako tvůrce Reactu, Facebook intenzivně využívá tuto knihovnu ve své webové aplikaci.
 2. **Instagram:** Instagram, který je součástí Facebooku, také využívá React pro své webové rozhraní.
 3. **Airbnb:** Tato platforma pro sdílení ubytování používá React pro vylepšení uživatelského rozhraní své webové aplikace.

##### Příklady využití

 1. **Jednostránkové aplikace (SPA):** React je ideální pro vývoj SPA, kde je potřeba dynamicky měnit obsah bez načítání nové stránky.
 2. **Mobilní aplikace:** S React Native, odvozeným produktem Reactu, mohou vývojáři vytvářet nativní mobilní aplikace pro iOS a Android pomocí JavaScriptu.
 3. **Webové aplikace s bohatým uživatelským rozhraním:** React je vhodný pro komplexní aplikace, kde je potřeba efektivní správa stavu a dynamické aktualizace UI.
