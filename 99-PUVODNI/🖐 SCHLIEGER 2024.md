# Dotace scripts
---
Vytvoř online python script spustitelný z telefonu android i osx pro výpočet dotace nový zelená úsporám.

Výše dotace se zkládá z:

- využití s čerpadlem : vybrat ano/ne  -> vzorec (NE +30.000 Kč, ANO +60000 Kč)
- výkon instalace: (bude doplněn) vzorec -> (doplněný výkon) krát 8.000 Kč

- kapacita baterií: (bude doplněna) vzorec -> doplněná kapacita krát 8.000 Kč

- pokud je PO SEČTENÍ těchto 3 částek hodnota vyšší než 160000 Kč - sniž hodnotu na 160000 Kč

Zjisti další proměnné a přičítej je k celkové výši dotace..
- připojení boileru min. 120l : vybrat ANO/NE -> vzorec (ANO +10.000 Kč)
- KOMBINAČNÍ BONUS FVE+ZATEPLENÍ : vybrat ANO/NE -> vzorec (ANO +30.000 Kč)

Výpočet zobrazit rozepsaný po jednotlivých částí viz. výše.
